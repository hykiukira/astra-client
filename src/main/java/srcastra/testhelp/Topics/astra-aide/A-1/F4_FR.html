F4 Touche de Chargement

La fonction chargement d'un objet est particulière. Elle sert à sélectionner un élément d'un tableau qui est visible sur l'écran. Sauf que dans certains cas, quand F4 n'est pas activée, cette fonction est remplacée par la touche F3 modification. Ce paradoxe est motivé par un gain de temps dans le cas où tous les éléments (champs) d'un tableau sont visibles
