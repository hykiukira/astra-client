F9 Touche de suppression

La fonction d'effacement a comme effet de supprimer logiquement des informations dans la base de données. Des contrôles automatiques se font dans ASTRA quant à l'utilisation de la touche F9.

Exemple : L'utilisation de F9 sur la ligne de valeur aura comme conséquence que ASTRA établira une note de crédit si le dossier modifié est déjà facturé. Dans d'autres cas la fonction vous préviendra qu'elle ne peut pas annuler telle information selon les liens de cette dernière avec d'autres éléments reliés à cette information.
