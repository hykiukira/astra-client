F5 Touche d'avancement et validation

La fonction d'avancement permet de valider vers la fin du scénario de création ou de modification. Elle permet également dans des cas précis de mettre à jour la base de données.

Exemple, dans la création d'un dossier, F5 suit le scénario de création programmé jusqu'à l'enregistrement d'un acompte, cette dernière phase abouti par un message qui vous prévient de l'enregistrement définitif dans la base de données.

Au contraire de F8 qui vous permet de reculer dans le scénario, bien que cette dernière fonction n' efface pas le contenu enregistré dans la base de données.
