F4 In-ladentoets

De functie laden van een voorwerp is bijzonder. Zij dient om een element van een tabel te selecteren die zichtbaar is op het scherm. Behalve dat in zeker geval, wanneer F4 niet wordt geactiveerd, deze functie door de toets F3 wijziging wordt vervangen. Deze paradox wordt voor een winst van tijd ingesteld in het geval of alle elementen (velden) van een tabel zichtbaar zijn.
