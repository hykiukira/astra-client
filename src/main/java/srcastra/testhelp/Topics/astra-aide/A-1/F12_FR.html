F12 Touche Navigateur

La fonction F12 permet de voyager dans les différents objets (sous fenêtres, panneaux) d'un produit vendu.
Ceci aussi bien en création qu'en modification d'un dossier.

Exemple : Le produit d'une brochure est composé de trois objets (sous écran).
Le premier est le général, le second est la définition de l'accommodation et le troisième la feuille de prix.

Cette fonction vous permet la navigation sans souris entre ces trois objets écran.
L'objet actif est encadré en bleu.
