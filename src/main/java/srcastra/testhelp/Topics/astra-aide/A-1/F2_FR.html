F2 Touche de création

La fonction F2 a pour but de permettre de créer des objets (données) qui n'existent pas encore.

Exemple F2 : Lors de la création d'un nouveau dossier. Si vous vous trouver dans un produit
hôtel, brochure, ou sur un billet dans le sous écran des lignes de prix ou encore sur le sous
écran de définition des accommodations, la touche F2 crée une nouvelle ligne d'information
de ce module sélectionné.
