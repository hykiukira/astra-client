F10 Touche d'assistance d'encodage des compléments de descriptions

Cette fonction, vous permet de sélectionner des descriptifs d'aide à l'encodage de textes.
Ces descriptifs sont spécifiques par catégorie de produit vendu.

Exemple : Brochure, aviation, hôtel, taxi etc.. ont pour chaque catégorie leur propre descriptif.

F10 s'active également lors du choix d'un code de TVA. Une assistance très utile.
