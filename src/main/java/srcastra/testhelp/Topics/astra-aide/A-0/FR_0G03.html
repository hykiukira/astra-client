 ASTRA est un logiciel conçu pour se maintenir à jour automatiquement, avec l’aide de WEB START, implanté sur le serveur de notre firme. Nuit et jour à votre disposistion.

WEB START inclut des dispositifs de sécurités.
Il contrôle l'intégrité du logiciel ASTRA.
 Vous utilisez le sytème technique à chaque chargement du programme.
WEB START contrôle votre version du logiciel et en corrige les anomalies éventuelles ou transformations de votre code.
En outre, la technologie ASTRA-WEB START vous permet d’utiliser la dernière technologie de maintenance dynamique.
WEB START fonctionne avec n’importe quel browser INTERNET, il vous suffit de cliquez une fois sur application dans notre site, pour que WEB START se mette en opérationnel. Après la 1ere installation une Icône apparaît sur le bureau de votre PC et ne nécessite plus l’usage du browser.
