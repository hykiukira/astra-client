WEB START
Sun Microsystems, Inc. 4150 Network Circle Santa Clara, CA 95054


Is een software systeem dat op de server van onze onderneming wordt gevestigd. Om zich aan dag ASTRA automatisch te handhaven.

WEB START heeft ingesloten een veiligheidsvoorzieningen voordat het controle van integriteit van uw programma's altijd kwitantie is.

WEB START controleert uw versie van de software en corrigeert de afwijkingen ervan.

WEB START laat u toe de laatste technologie ASTRA in gebruik te hebben met hulp van een dynamisch onderhoud technologie.

Installatie :
WEB START opladen van uit onze site, en hem installeren.
Het voldoet u van een keer te kliken op de toepassing ASTRA van uit u internet browser voor dat WEB START zich automatich operationeel zet.
Na eerste de installatie is er een Icône op het burreel blad van uw PC aangemaakt, en vereist het gebruik van de browser niet meer.
