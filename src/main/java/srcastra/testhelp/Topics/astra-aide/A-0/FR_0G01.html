ASTRA Concepts techniques XinOne

Logiciel d'une nouvelle gamme, basé sur le système d'architecture XinOne. Ce qui signifie que tous les processus sont intégrés dans l'application ASTRA.

ASTRA utilise une seule Base de Données, une seule technologie d'accès au réseau Intranet ou Internet, une seule table de transactions de gestion en valeures pour les dossiers : Financières, Ventes, Achats, Opérations de régularisation, Opérations de Banques.

XinOne est un concept innovateur et rapide qui facilite la tenue de la comptabilité, la gestion multi-bureaux.

XinOne supprime le concept du lien comptable ainsi que le transfert de nuit dans le cas de gestion multi-bureaux. Toutes les données sont mises à jours dans tout le système ASTRA au même moment.

Les rapports et les analyses, situation d'encaissement reflètent toujours les situations les plus récentes.
XinOne intègre automatiquement les factures d'achats de certains Tour-operators ainsi que BTN.
XinOne intègre les données des CDS tels Galiléo, Amadeus, Worldspan dans le cas d'une Agence IATA.
