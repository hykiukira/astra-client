ASTRA est conçu avec l'aide d'un langage qui se veut sécurisé.

Sécurité pour quoi et contre quoi ?

La caractéristique qui attire particulièrement l'attention est que ASTRA est un programme dynamiquement portable, donc multi-plateformes. Par son langage ASTRA fournit plusieurs couches de protections. Ces couches de protections se protègent contre les codes dangereusement endommagés, ou contre les choses plus vicieuses telles que certains types de virus.

La machine virtuelle est un mur de protection autour d'ASTRA. Ce mécanisme est la fondation d'une politique de sécurité de haut niveau qui autorise ou interdit plusieurs types d'activités en fonction des tâches que ASTRA doit effectuer.
