INTRODUCTION

 Ce produit se base sur un outil de développement d'une puissance exceptionnelle pour développer des applications compatibles avec différents operating systems tels que WINDOWS, MACINTOCH, LINUX, UNIX.
Que votre serveur soit LINUX, WINDOWS le logiciel ASTRA SERVER s'adapte. Que vos stations de travails soient WINDOWS, LINUX ou MACINTOCH le logiciel ASTRA CLIENT s'adapte.

Il est également transparent pour ADSL,SDSL, câble etc. . ., compatible avec les réseaux privés.

Le langage utilisé est sécurisé, il se protège contre la plus part des intrusions externes de codes pirates.

ASTRA TRAVEL BACK-OFFICE SYSTEMS est capable de se mettre à jour automatiquement. Corrections éventuelles, améliorations faites par les développeurs.
