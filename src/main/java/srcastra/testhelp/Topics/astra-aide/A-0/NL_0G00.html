INLEIDING

Deze software is opgericht met een uitzonderlijke en krachtige werktuigen voor ontwikkeling. Zo wel, dat ASTRA met andere operating systemen verenigbaar is, zoals WINDOWS, MACINTOCH OS, LINUX, UNIX, en dit zonde bijkomende kost.
Dat uw server werkt in LINUX of in WINDOWS NT technologie, de software ASTRA-SERVER past zich automatisch aan. Dat uw werkposten werken in WINDOWS, LINUX of in MACINTOCH/OS de software ASTRA-CLIENT past zich automatisch aan.

TELECOMMUNICATIE :
Doorzichtig voor ADSL, SDSL, KABELS, enz. Overeenkomend met privé netten zoals : BILAN.
