GESTION DOSSIER
ASTRA ECRAN DE BASE

Cette fenêtre se compose de deux objets (sous fenêtre) ou deux panneaux.

Partie inférieure, encadrée en bleu est active. Cette partie permet l'enregistrement des critères de recherches pour des dossiers existants. La partie supérieure affiche les résultats des recherches effectuées. Le résultat de la recherche est établi sous forme d'un tableau, qui reprend dans un ordre chronologique les dossiers triés sur l'argument de recherche.

Création d'un nouveau dossier :
Dans le cas ou vous voulez créer un nouveau dossier et non modifier un existant, il vous suffit de faire F2, le scénario création d'un dossier est donc défini.

Modification d'un dossier :
Dans le cas contraire, où le choix est de modifier un dossier existant. Il faudra rechercher le dossier par un des arguments de recherches. La touche F12 vous permet de naviguer sur les arguments de recherches de l'un à l'autre.
Autres particularités dans la recherche, est que le tableau inférieur de recherche vous permet de faire une combinaison entre plusieurs arguments de la colonne droite (ex :numéro dossier) et ceux de la colonne gauche (ex :date de départ). Après sélection de l'un d'eux ou d'une combinaison, le tableau supérieur est chargé avec les dossiers sélectionnés à l'aide de F5 ou de la touche ENTER.

Attention :
 il vous faut dès lors sélectionner le dossier dans le tableau à l'aide de la flèche haut ou bas et utiliser la touche F4 pour charger en mémoire, le scénario modification dossier sera donc sélectionné.  

Attention :
La touche F11 vous permet de quitter ce module.
