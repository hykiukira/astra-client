GESTION DOSSIER
FENETRE DEFINTION DU CLIENT

Cette fonction comprend trois objets actifs (panneaux) et deux objets neutres.

Panneaux Actifs :
Le panneau supérieur permet la signalisation de la date du départ et date du retour pour l'ensemble du forfait vendu. Une fonction calcule automatiquement le nombre de jour total du forfait et le nombre de nuités. Cette dernière information est modifiable.
Ce panneau comprend ensuite le champ TYPE qui correspond au type de vente que vous réalisez.
Particularités du champ TYPE:
Selon qu'une interface d'intégration existe par exemple BTN ou Worldspan, Xinone vous permet l'intégration automatique des données GDS. Donc il est nécessaire de faire attention dans le choix que vous effectuez.

Le champ USER est chargé automatiquement par le code utilisateur.

Le champ CLIENT CONTRACTUEL et CLIENT FACTURABLE vous permettent de sélectionner dans la base de données les noms en conséquence de votre choix.
Particularités de ces deux champs:
Dans le cas où le client n'existe pas encore, il suffira de cliquerà l'aide de la souris sur la fenêtre gestion des clients, qui s'est activée des l'ouverture du module gestion de dossier. F2 et vous pouvez créer un client.
Attention pour ceux qui n'aime pas la souris faite le avec la combinaison CONTROL+F12.

Le panneau du milieu permet par deux boutons d'obtenir les coordonées complètes du client contractuel ou facturable selon le besoin.

Le panneau inférieur ou panneau des passagers comporte des particularités dans le scénario de création d'un nouveau dossier.
Particularités panneau passagers
Lorsque le panneau passagers est vide, F2 ouvre un objet-fenêtre dans lequel vous pouvez choisir plusieurs options.
Options liste des passagers vous permet d' entrer individuellement les passager.
Option groupe vous permet d'entrer un nom unique du groupe.
Option à cocher enregistre automatiquement le client contractuel en tant que passager.
Option à cocher qui inclu les passagers du dernier dossier exemple d'une famile.
Attention F5 est activée en rouge immédiatement, ce qui veut dire que vous pouvez utiliser cette fonction directement dans le cas où les autres champs ne vous interessent pas.

Panneau neutres :
Les panneaux neutre comporte les informations récapitualtives d'aide.

Le panneau supérieur rappel en permanence le numéro de dossier, date de création, date de départ, de retour, le statut général du dossier, le type de booking, le nom du client contractuel.
Particularité du statut :
Le statut général du dossier prend la valeur de OK, si tous les produits vendu sont OK dans le cas contraire il reçevra la valeur d'un status différent.

Le panneau inférieur est le rappel récapitulatif en valeur total du dossier.
