GESTION DES DOSSIERS
PRODUIT BROCHURE

Cette fenêtre comprend trois panneaux. Comme pour les autres types de ventes (groupe de produit) qui peuvent composé un dossier (ex: avion, hôtel). Cette fenêtre comprend uniquement les informations du groupe produit sélectionné ou créé.
Si votre feuille de prix comprend plusieurs groupes de produits, en cas d'organisation personnelle d'un voyage, cette fenêtre ne comprend qu'un seul groupe. Il vous faudra retourner vers la feuille de prix générale (F5) pour sélectionner (F4) ou créer (F2) un autre groupe de produit. En cas que vous sélectionnez un groupe déjà crée, la touche navigation (F12) vous permettra de sauter d'un panneau vers l'autre.

Panneau inférieur

Comprend 2 panneaux qui se superposent, le premier est la définition de la chambre ainsi que les commodités en générale. Le deuxième est la feuiile de prix spécifiques à ce groupe produit. (F12) vous permet de naviguer entre les panneaux, (F5) valide celui qui est actif, (F8) permet de sortir de celui qui actif sans le modifier.

Panneau inférieur feuille de prix

La feuille de prix se compose de 3 types de prix. Le prix de base, les suppléments et les réductions. Dans le cas ou ASTRA vous présente une feuille de prix qui est vide, il désidera de lui même que le premier prix doit obligatoirement être le prix de base. Il est important de savoir que astra ne vous laissera pas vendre un supplément ou réduction dans le cas ou il n'exite pas de prix de base.

Une vue d'ensemble en forme de tableau représente les prix enregistrés, en bleu la ligne qui pourrait être selectionné par (F3) en cas de modification. (F2) vous permet de créer une nouvelle ligne de prix. Facile à la lecture le panneau prix vous permet d'avoir la vue sur vos obligation financière comme par exemple, la TVA qui serait comprise ou nom dans le prix.
