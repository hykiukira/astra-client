GESTION DOSSIERS
FEUILLE DE PRIX

Cette fenêtre comprend 3 panneaux. Le panneau supérieur comprend 3 champs actifs qui permettent de créer des nouvelles lignes de produits dans la feuille de prix. Le panneau central est le tableau feuille de prix des produits vendus ( pour les utilisateurs de Voyager's ceci est équivalent aux ACTIONS ). Le panneau inférieur de cette fenêtre comprend 5 champs actifs pour la gestion des documents à recevoir des fournisseurs, cette gestion de documents est liès à l'application AGENDA des tâches à éffectuer.

Panneau de sélections
Les champs de sélections ne servent que lors de la création d'une nouvelle ligne de produit, F2 actionne la fonction de création. Dans le cas d'une feuille de prix vide F2 est automatique.

Type de Produit : permet la sélection d'un type de vente Brochure, Avion, Bateau, Taxi etc. . .

Fournisseur: permet de sélectionner le fournisseur que vous vendez.

Groupe de produit: permet l'association de cette vente au groupe de produit.
Particularités du champ Groupe de produit:
Voir dans l'aide fournisseur module groupe de produit ainsi que la table de décision

Panneau feuille de prix
Les produits créés dans ce panneau peuvent être facilement sélectionnés à l'aide des flèche bas ou vers le haut du clavier ou encore avec l'aide de la souris. La touche F4 permet de séléctionner la définition détaillé de ce produit et de le modifier.
A l'extrème droite un champ à cocher est prévu pour l'annulation de lignes de prix (Voir aide Annulations).

Panneau de gestion de documents à reçevoirs
Lors de la création d'un fournisseur ASTRA, vous avez la posssiblité de définir le nombre de document à reçevoir de ce fournisseur. Ce panneau vous permet d'actualiser les documents entrées.
Particulartés pour y accéder :
F3 modification permet l'accès à ce module document.

Récapitulation
F2 permet la création d'un nouveau groupe fournisseur une action de vente.
F3 permet d'accéder au panneau des document à reçevoir.
F4 permet la sélection d'un groupe de produit.
F5 permet l'avancement du scénario de création ou de modification du dossier vers les paiements.
F8 permet de reculer vers l'écran de définition de client ou passagers.
F9 permet l'annulation de ligne ou du dossier.
