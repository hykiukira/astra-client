ASTRA TRAVEL BACK-OFFICE SYSTEMS


Logiciel d'une nouvelle gamme, basée sur le système d'architecture XinOne. Ce qui signifie que tous les
processus sont intégrés. Vous n'utilisez désormais plus qu'une seule base de données, une seule technologie d'accès au réseau Intranet ou Internet, une seule table de transactions pour les dossiers, financiers, ventes, achats, opération de régularisation.
XinOne est un concept innovateur, rapide et convivial, facilite la tenue de la comptabilité, la gestion multi-bureaux. Plus de lien comptable, transfert de nuit. Toutes les données sont mises à jour dans tous le système au même moment. Les rapports et les analyses reflètent toujours la situation la plus récente.
Ce produit se base sur des outils de puissance exceptionnels pour développer des applications compatibles avec différents operating systems tel que WINDOWS, MACINTOCH, LINUX, UNIX ceci sans qu'il vous en coûte. Il est transparent pour ADSL,SDSL, câble etc…, compatible avec les réseaux privés exemple : BILAN de la firme BELGACOM.
